# Basra 
An online Egyption Card Game

## Features
- Chat.
- Up to 4 players.
- Card back and background shops.
- Friends System.
- Challenge Friends.
- Leveling System.
- If you got disconnected, you can resume the same room, and your missed turns are played randomly.
